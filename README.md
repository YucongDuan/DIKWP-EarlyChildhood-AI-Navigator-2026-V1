# DIKWP EarlyChildhood AI Navigator 2026 V1

A standalone, offline-first prototype for supporting 0–6 year-old child development in the AI era. It is designed for parents, grandparents, caregivers, nursery providers, kindergarten teachers, community child-health staff, and managers.

## Workflow

Child profile → age-band development map → DIKWP D/I/K/W/P/R growth loop → nurturing care plan → serve-and-return play studio → AI/screen guardrails → health/nutrition/sleep/safety → observation evidence ledger → family/nursery/kindergarten/child-health collaboration → Action Tickets → Child Development Passport.

## Boundary

The system does not diagnose, rank, label, or certify children. It does not replace pediatricians, child-development specialists, teachers, caregivers, or parents. It does not store real photos, video, biometric data, medical records, or location data.

## Quick start

Open `index.html` in a modern browser. No server, database, account, API key, or network is required.

Optional CLI:

```bash
python tools/run_child_navigator.py examples/sample_child_profile.json --out outputs
```
